GitHub PR Review Tracker

Configure your GitHub Personal Access Token

Create a token at github.com/settings/tokens with repo scope (or public_repo for public repos only).

How to get a token

  1. Go to GitHub Token Settings
  2. Give it a name like "PR Review Tracker"
  3. Select the repo scope (or public_repo for public repos only)
  4. Click "Generate token"
  5. Copy the token and paste it above
